# AI-Driven HR Recruitment Analytics and Candidate Selection System

[Project Status: Production-Ready] [Category: HR Tech] [Focus: Recruitment Automation]

This project demonstrates a working implementation of AI-based recruitment analytics using machine learning and Power BI dashboards.

---


## 1. Project Overview

This project demonstrates the application of Artificial Intelligence in Human Resource Management, specifically in the recruitment process. It integrates machine learning for candidate selection prediction and Power BI for interactive data visualization.

The system simulates real-world HR analytics used by organizations to improve hiring efficiency and decision-making.

---

## 2. Objectives

* To analyze recruitment data using data analytics techniques
* To predict candidate selection using machine learning
* To visualize hiring insights through an interactive dashboard
* To demonstrate AI-driven decision-making in HR

---

## 3. Key Features

### 3.1 Recruitment Analytics Dashboard

* Total Candidates Overview
* Selected Candidates Count
* Selection Rate Analysis
* Country-wise Hiring Trends
* Experience Impact on Hiring
* Skill vs Technical Performance
* Salary Distribution Analysis

### 3.2 Machine Learning Model

* Predicts candidate selection based on:

  * Skill Score
  * Technical Test Score
  * Communication Score
  * Experience
* Built using classification algorithms

### 3.3 Interactive Filtering

* Country
* State
* Gender
* Education

## 6. Methodology

### Step 1: Data Collection

A structured dataset containing candidate information such as demographics, skills, experience, and test scores was used.

### Step 2: Data Preprocessing

* Data cleaning
* Handling missing values
* Feature selection

### Step 3: Model Development

* Applied classification algorithm
* Trained model on recruitment dataset
* Generated predictions for candidate selection

### Step 4: Dashboard Development

* Imported dataset into Power BI
* Created KPIs and measures
* Designed interactive visuals
* Applied filters and slicers

* Hiring trends vary across different countries and regions
* Salary expectations influence selection outcomes
